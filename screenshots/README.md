# Screenshots — QA visuelle Plinko

Captures datées des états validés du jeu.

## Convention de nommage

```
YYYY-MM-DD_description-courte.png
```

Exemples :
- `2026-03-29_plateau-config-validee.png`
- `2026-03-31_overlay-recompense.png`
- `2026-04-01_visuel-end-game-jackpot.png`

## Workflow

Les captures sont effectuées depuis **Chat (Claude.ai)** avec partage de screenshot.
Ce dossier est le bridge entre les sessions design (Chat) et le code (Claude Code).
